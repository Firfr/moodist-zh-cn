# 本地调试命令记录

这里记录除了汉化之外的代码修改

## 运行环境
node: `20.19.1-alpine3.21`

## 开发运行

- 容器运行
  ```bash
  docker run --rm -it --name Node_Code \
  --cpus=2 --memory=4G --network host \
  -w /app -v .:/app node:20.19.1-alpine3.21 ash
  ```
- NPM 换源
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```
- 安装依赖
  ```bash
  npm i
  ```
- 运行
  ```bash
  npm run dev
  ```
- 编译
  ```bash
  npm run build
  ```
